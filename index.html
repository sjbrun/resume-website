    Current Data Analysis Program

    Link to project

    This program compiles and analyzes temperature, voltage, and current test data of complex automotive lighting systems. Raw data (txt and csv files) is translated to dataframes (pandas module objects) and organized using an object-oriented approach. From there the data objects are analyzed primarily with help from matplotlib. I created a simple GUI (pyqt5 module) to enable my coworkers without programming/scripting experience to use this valuable analysis tool.

    Holiday Karaoke Madlibs Application

    Link to project (may take a few seconds to open)

    This is a Madlibs-inspired karaoke app I created for a holiday party I threw with friends in December 2016. Aesthetically I was going for the website version of an ugly Christmas sweater! Users pick a holiday song from the list and have the choice of singing the original lyrics or creating their own. If creating their own lyrics, users are prompted to provide a series of words that will replace certain words in the chosen song (a la Madlibs). After user entry is complete, an audio karaoke track of the song is provided and the lyrics are displayed.
